# ADK_Basic Agent Project

This project demonstrates a basic implementation of an AI agent using the Google Agent Development Kit (ADK).

## Project Overview
The goal of this project is to build and orchestrate an AI agent capable of handling natural language requests, utilizing the Gemini API for reasoning and task management.

## Features
- **Agent Orchestration:** Uses the Google ADK framework to manage agent lifecycle.
- **Interactive Web UI:** Includes a local web interface for testing agent reasoning and traces.
- **Traceability:** Real-time visibility into the agent's thought process.
